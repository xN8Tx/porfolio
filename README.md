## 🤔 About

This portfolio site aimed at showcasing my skills and list of my projects.

Client side based on NextJS and TypeScript.All animation on this project created by LocomotiveScroll and GSAP. This site has a `next-intl` on board, including Russian and English.

Server side based on Strapi.
